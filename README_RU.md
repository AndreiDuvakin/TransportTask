# Транспортная задача: дельта-метод (Delta Method)

## О проекте

Данный проект разработан в рамках учебной практики по специальности **09.02.07 Информационные системы и программирование** (ПМ.02 Осуществление интеграции программных модулей). Программа реализует решение классической транспортной задачи с помощью **дельта-метода** (комбинация метода северо-западного угла и метода потенциалов).

Проект предназначен для оптимизации плана перевозок от поставщиков (складов) к потребителям (магазинам) с минимальными суммарными затратами.

## Цель создания

- Получение практического опыта интеграции программных модулей.
- Реализация экономико-математической модели на практике.
- Демонстрация навыков:
  - разработки требований к программным модулям;
  - написания и отладки кода;
  - создания тестовых сценариев;
  - инспектирования кода на соответствие стандартам.

## Что делает программа

1. Принимает входные данные:
   - количество поставщиков и потребителей;
   - запасы (склады) и потребности (магазины);
   - матрицу стоимостей перевозок.
2. Автоматически генерирует начальную матрицу (возможен ручной ввод).
3. Решает транспортную задачу **дельта-методом**.
4. Выводит:
   - оптимальный план перевозок;
   - минимальную общую стоимость перевозок.

Программа также сообщает, если задача не может быть решена данным методом.

## Алгоритм (основные шаги)

1. Преобразование матрицы стоимости → таблица приращений.
2. Поиск ячеек с единственным нулевым приращением.
3. Построение и оптимизация плана перевозок.
4. Перераспределение поставок через цепочки.
5. Проверка оптимальности методом потенциалов.
